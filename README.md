
# Download Simon Stålenhag images

The `StalenhagDownloader` is a Python class which downloads all images from the website [simonstålenhag.se](https://www.simonstalenhag.se/). Using threading, the downloading of images is significantly faster than before.


## Made by

- [@RasseTheBoy](https://github.com/RasseTheBoy)

## Download location

`start()` method initiates the download process and creates a new directory with the path provided by `main_dir_path`. The images will be downloaded where the python file is located, if no path is given to `main_dir_path`.
